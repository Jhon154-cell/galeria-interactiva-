# Galería Interactiva con JavaScript

Este proyecto consiste en una galería de imágenes interactiva desarrollada con HTML, CSS y JavaScript puro, aplicando manipulación del DOM y manejo de eventos.

## Funcionalidades
- Agregar imágenes mediante URL
- Seleccionar una imagen (solo una a la vez)
- Eliminar la imagen seleccionada
- Uso de eventos de teclado y mouse

## Tecnologías
- HTML5
- CSS3 (Grid)
- JavaScript (DOM)

El proyecto fue publicado utilizando GitHub Pages.
